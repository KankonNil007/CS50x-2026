## Overview

This problem set dives into low-level memory management in C, including pointers, dynamic allocation, and file I/O.

## Problems

### 1. Volume

A program to modify the volume of an audio file (`.wav`) by reading samples from a file, scaling them, and writing them to a new file.

- **Folder**: `01-Volume`

### 2. Filter

A program that applies filters to BMP images, manipulating individual pixels.

- **Filter (Less)**: Implements Grayscale, Sepia, Reflection, and Blur.
- **Filter (More)**: Adds edge detection.
- **Folders**: `02-Filter-Less`, `03-Filter-More`

### 3. Recover

A program that recovers JPEGs from a forensic image (raw memory card dump) by looking for JPEG signatures and analyzing memory blocks.

- **Folder**: `04-Recover`
