## Overview

This problem set introduces SQL (Structured Query Language) for storing, querying, and managing relational databases.

## Problems

### 1. Songs

A program/lab to answer questions about a database of songs and artists (`songs.db`) using SQL queries.

- **Folder**: `01-Songs`

### 2. Movies

A series of SQL queries to answer questions about the IMDb movie database (`movies.db`), linking movies, stars, directors, and ratings.

- **Folder**: `02-Movies`

### 3. Fiftyville

A detective mystery! Using SQL queries on a database of crime scene reports, interviews, bank transactions, phone calls, and flights (`fiftyville.db`) to identify a thief and their accomplice.

- **Folder**: `03-Fiftyville`
