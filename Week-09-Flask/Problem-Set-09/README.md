## Overview

This problem set combines front-end and back-end skills to build dynamic web applications using Python, Flask, and SQL.

## Problems

### 1. Birthdays

A web application to keep track of friends' birthdays.

- Allows users to add birthdays via a form (POST) and view listed birthdays (GET).
- Interacts with a SQLite database.
- **Folder**: `01-Birthdays`

### 2. Finance

A stock trading simulator.

- **Authentication**: Implementing register, login, and logout.
- **Quoting**: Fetching real-time stock prices.
- **Buying/Selling**: Handling "fake" cash transactions and updating user portfolios.
- **History**: Tracking all transactions.
- **Folder**: `02-Finance`
